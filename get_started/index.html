Launching an experiment is fast and simple
4 easy steps to run your code on nodes

1. Build your application

Build one or several firmwares to use during your experiment for WSN430, M3 or A8 hardware platform.

2. Select resources

Choose number and type of nodes or choose a specific topology.

3. Configure nodes

Associate a firmware to each node and define the activity of the control node during the experiment

4. Submit your experiment

A scheduler will program your experiment, immediately if resources are free. The deployment will be automatic.
